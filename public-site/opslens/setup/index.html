Setup guide

app-sync integrations / OpsLens / Setup

Install and configure OpsLens for HubSpot with confidence.

This setup guide is specific to the OpsLens HubSpot integration. It walks through prerequisites, plan expectations, the install flow from apps.app-sync.com, HubSpot OAuth authorization, and what your team should see after the connection is complete.

View install-complete page Contact support

Prerequisites

  • A HubSpot portal where you can install connected apps
  • Access to approve the OpsLens OAuth authorization step
  • Your alert destinations and workflow owners ready to confirm after install

Plan note

  • OpsLens access is tied to your selected app-sync plan during install
  • Plan and billing details appear again on the install-complete page
  • For billing questions, contact billing@app-sync.com before rollout

Before you start

Make sure the right HubSpot admin is doing the install.

OpsLens should be installed by the person who can approve connected app access for the target HubSpot portal and confirm the team’s alerting expectations afterward. If you also plan to configure Slack notifications, have the destination webhook ready before you open Settings in HubSpot.

Step 1

Start the install from apps.app-sync.com.

Begin from the OpsLens public app entry on apps.app-sync.com and choose the install path for your HubSpot portal. This is the public starting point you can share during marketplace review or customer onboarding.

Step 2

Approve the HubSpot OAuth authorization.

HubSpot will prompt the installer to authorize OpsLens for the required portal. Review the requested permissions, confirm the correct portal is selected, and finish the OAuth approval flow to complete the connection.

Step 3

Confirm the install-complete handoff.

After OAuth succeeds, OpsLens returns the installer to the external install-complete page. There you should see the portal ID, plan, billing interval, bootstrap status, and the links that open OpsLens in HubSpot or point you to the fallback HubSpot path.

What you should see after install

A successful install looks explicit, not silent.

  • An install-complete confirmation page for OpsLens
  • The connected portal ID and selected plan information
  • A bootstrap status that indicates the install is ready or in progress
  • Direct HubSpot actions when app-specific deep links are available

Find OpsLens in HubSpot

Use App Home first, then Settings.

Once the install is complete, open HubSpot and go to Connected Apps, then select OpsLens. From there, use App Home to review the dashboard and use Settings to configure portal-specific options such as alerts and critical workflow behavior.
